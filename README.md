# Sales-Dashboard-for-ABC-Company
1. Project Title / Headline

Sales Performance Dashboard for ABC Company (2020–2022) | Power BI Business Intelligence Solution

2. Short Description / Purpose

This Power BI dashboard provides a comprehensive analysis of ABC Company's sales performance from 2020 to 2022. The dashboard helps stakeholders monitor sales, costs, product performance, supervisor-wise performance, brand contribution, and inventory trends through interactive visualizations and KPI metrics. It enables data-driven decision-making by transforming raw sales data into actionable business insights.
